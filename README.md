## Plugin manager

Requires [lazy.nvim](https://github.com/folke/lazy.nvim) as the plugin manager.

## Configured language servers

- Python (jedi)
- Lua (luals)
- Tex (texlab)
- Vim (vimls)
- Bash (bashls)
- C/C++ (clangd)
- CSS/SCSS/LESS (cssls)
- Markdown (marksman)
